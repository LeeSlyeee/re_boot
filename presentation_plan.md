# Re:Boot 기획 발표 자료

## 1. 프로젝트 개요 (Intro)

### **슬로건**

> **"모든 학생에게 전담 AI 과외 선생님을 배정하다"**
> **놓치는 내용 없이 200% 소화하는 AI 기반 초개인화 학습 보조 플랫폼**

### **기획 배경 (Problem)**

- **진도 격차**: 강의 중 잠깐만 딴생각해도 진도를 놓침.
- **필기 부담**: 필기하느라 정작 중요한 설명을 듣지 못하는 주객전도 발생.
- **질문 장벽**: 수업 흐름을 끊을까 봐 질문하지 못하고 넘어가는 경우가 많음.
- **복습의 어려움**: 1시간짜리 강의 영상을 다시 돌려보려면 너무 많은 시간이 소요됨.

---

## 2. 핵심 기능 소개 (Key Features)

### **기능 1: 실시간 AI 노트 (Real-time Lecture Note)**

> **"필기는 AI가, 나는 이해에만 집중"**

- **실시간 STT (Speech-to-Text)**
  - OpenAI Whisper 모델을 활용하여 강의 내용을 실시간으로 텍스트 받아쓰기.
  - 마이크(오프라인), 유튜브 URL, PC 시스템 사운드(Zoom/인강) 등 다양한 소스 지원.
- **스마트 요약 (Smart Summarization)**
  - 단순 받아쓰기를 넘어, AI가 **핵심 문맥을 파악하여 요약**.
  - 잡담, 추임새("음..", "저기..") 자동 필터링.
  - 마크다운(Markdown) 형식으로 깔끔하게 구조화된 노트 자동 생성.

### **기능 2: 나만의 AI 튜터 (Context-Aware AI Tutor)**

> **"언제든 물어보세요, 문맥을 아는 AI니까요"**

- **RAG (검색 증강 생성) 기반 챗봇**
  - 일반 ChatGPT와 달리, **"오늘 수업한 내용"**을 정확히 기억하고 답변.
  - "아까 교수님이 말한 그 예제, 다시 설명해줘" 같은 질문 가능.
- **세션 격리 (Session Isolation)**
  - 학생 개개인의 질문과 학습 데이터가 서로 섞이지 않도록 **완벽한 데이터 격리** 구현.

### **기능 3: 데일리 퀴즈 & 복습 (Daily Quiz System)**

> **"오늘 배운 내용, 오늘 내 것으로"**

- **자동 퀴즈 생성**
  - 수업이 끝나면 AI가 강의 내용을 바탕으로 3~5개의 퀴즈를 즉석에서 출제.
  - 단답형, 객관식 등 다양한 유형 제공.
- **피드백 시스템**
  - 채점 결과 즉시 확인 및 오답에 대한 상세한 해설 제공.

### **기능 4: 프리미엄 학습 대시보드 (Glassmorphism UX)**

- **몰입형 디자인**: 글래스모피즘(Glassmorphism) UI 적용으로 세련되고 직관적인 학습 환경 제공.
- **학습 현황 시각화**: 오늘 수강한 수업, 퀴즈 점수, 학습 누적 시간 등을 한눈에 파악.

## 3. 시스템 아키텍처 및 기술 스택 (Architecture)

| 영역         | 기술 스택                   | 비고                            |
| :----------- | :-------------------------- | :------------------------------ |
| **Frontend** | React / Vue.js 3, Vite      | Glassmorphism CSS 적용          |
| **Backend**  | Django REST Framework (DRF) | Python 기반의 안정적인 API 서버 |
| **AI Core**  | OpenAI GPT-4o, Whisper      | STT 및 요약/생성 모델           |
| **Database** | PostgreSQL (pgvector)       | 벡터 검색(RAG) 지원             |
| **Infra**    | Docker                      | 컨테이너 기반 배포 환경         |

---

## 4. 기대 효과 (Impact)

1. **학습 효율 극대화**: 필기 노동에서 해방되어 강의 내용 자체에 100% 집중 가능.
2. **복습 시간 단축**: 1시간 강의를 5분 요약 노트와 퀴즈로 빠르게 리마인드.
3. **낙오자 없는 교육**: 이해하지 못한 부분은 AI 튜터가 1:1로 끝까지 케어.

---

## 5. 시연 시나리오 (Demo Flow)

1. **로그인**: 학생(Student) / 강사(Professor) 계정 로그인.
2. **수업 시작 (학생)**:
   - '실시간 학습' 버튼 클릭.
   - 마이크 권한 허용 후 강의 시작 (혹은 유튜브 링크 입력/테스트 음성 재생).
   - 실시간으로 자막이 올라오는 모습 시연.
3. **AI 노트 확인**: 수업 종료 후 생성된 요약 노트 확인.
4. **AI 튜터 질문**: "오늘 배운 Vue.js 라이프사이클이 뭐야?" 질문 후 답변 확인.
5. **퀴즈 풀이**: 생성된 데일리 퀴즈 풀이 및 채점 결과 확인.
6. **대시보드**: 학습 기록이 갱신된 메인 화면 확인.
