## 📋 증분 구현 Step (우선순위순)

### 🔴 Step A: QR코드 + 퀴즈 타이머 + 펄스 경고 (UI 보완)

**소요: ~40분 | 임팩트: 높음 (즉시 눈에 보이는 개선)**

1. **교수자 QR코드**: 세션 코드를 QR로 변환하여 대시보드에 표시
2. **학습자 QR 스캔**: 모바일 카메라로 QR 스캔 입장
3. **퀴즈 타이머**: LiveQuiz에 `time_limit` 필드 추가 + 학생 팝업에 카운트다운
4. **펄스 50% 경고**: 이해도 50% 미만 시 빨간색 경고 표시

### 🟠 Step B: STT 키워드 스팟팅 + 스마트 퀴즈 제안 (핵심 차별점)

**소요: ~50분 | 임팩트: 매우 높음 (명세의 핵심 기능)**

1. STT 청크 수신 시 키워드 매칭 (`이해되셨나요`, `질문 있나요`, `다음으로 넘어갈게요`)
2. 키워드 감지 → AI 퀴즈 후보 자동 생성 (백그라운드)
3. 교수자에게 "퀴즈 준비됨, 발동할까요?" 스마트 팝업 전송
4. 교수자 "확인" → 즉시 전체 학생에게 퀴즈 발동

### 🟡 Step C: 학생용 Q&A 패널 + 유사 질문 묶음

**소요: ~40분 | 임팩트: 중간**

1. 학습자 화면에 Q&A 패널 추가 (다른 학생 질문 보기 + 공감)
2. 직접 질문 입력 (챗봇과 별개로 Q&A 전용 입력창)
3. 교수자 답변 → 학생에게 푸시
4. AI 유사 질문 클러스터링 (GPT로 그룹핑)

### 🟢 Step D: 세션 인사이트 리포트 + 개인 요약

**소요: ~50분 | 임팩트: 높음 (교수자 가치)**

1. 종료 시 **교수자 인사이트 리포트** 자동 생성:
   - 이해도 최저 구간 (펄스 히스토리)
   - 가장 많이 든 질문 TOP 3
   - 퀴즈 정답률 추이
2. **학생 개인 요약**: 퀴즈 통과율, 오늘 어려웠던 개념

### 🔵 Step E: 교안 매핑 + 교수자 승인 + 결석생 (고급)

**소요: ~1시간 | 임팩트: 높음 (완성도)**

1. IntegratedNote → 교안 + STT 통합 (교안 섹션별 매핑)
2. 교수자 검토 + 수정 화면
3. 승인 버튼 → 학생 배포 (전체/출석자/결석생 분기)
4. 결석생 → 노트 수신 + 셀프 테스트 플로우
