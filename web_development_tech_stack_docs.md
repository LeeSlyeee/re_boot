# 주요 웹 개발 IT 기술 스택 공식 문서 링크 모음

현대 웹 개발에서 널리 사용되는 주요 기술, 프레임워크 및 도구들의 공식 문서 페이지 링크 모음입니다. 각 카테고리별로 핵심 기술들의 원본 문서를 참고하실 수 있습니다.

## 1. 🎨 프론트엔드 (Frontend) - 사용자 영역

- **HTML & CSS & JavaScript (MDN Web Docs)**: 웹의 가장 기본이 되는 기술의 사실상 표준 기술 문서입니다.
  - [https://developer.mozilla.org/](https://developer.mozilla.org/)
- **TypeScript**: JavaScript에 정적 타입을 추가한 언어 (Microsoft).
  - [https://www.typescriptlang.org/docs/](https://www.typescriptlang.org/docs/)
- **React.js**: 사용자 인터페이스를 만들기 위한 JavaScript 라이브러리 (Meta).
  - [https://react.dev/](https://react.dev/)
- **Vue.js**: 사용자 인터페이스 빌드를 위한 프로그레시브 프레임워크.
  - [https://vuejs.org/guide/introduction.html](https://vuejs.org/guide/introduction.html)
- **Svelte**: 런타임이 아닌 빌드타임에 작업을 수행하는 컴파일러 기반 프레임워크.
  - [https://svelte.dev/docs](https://svelte.dev/docs)
- **Next.js**: 빠르고 SEO에 최적화된 React 풀스택 프레임워크.
  - [https://nextjs.org/docs](https://nextjs.org/docs)
- **Nuxt.js**: 직관적이고 강력한 Vue.js 기반 웹 프레임워크.
  - [https://nuxt.com/docs](https://nuxt.com/docs)
- **Redux**: JavaScript 앱을 위한 예측 가능한 상태 컨테이너.
  - [https://redux.js.org/](https://redux.js.org/)
- **Zustand**: 작고 빠르고 확장 가능한 상태 관리 솔루션 (React).
  - [https://zustand-demo.pmnd.rs/](https://zustand-demo.pmnd.rs/) (Github: [https://github.com/pmndrs/zustand](https://github.com/pmndrs/zustand))
- **React Query (TanStack Query)**: React 앱을 위한 강력한 비동기 상태(데이터 패칭) 관리 도구.
  - [https://tanstack.com/query/latest/docs/react/overview](https://tanstack.com/query/latest/docs/react/overview)
- **Tailwind CSS**: 마크업(HTML) 내에서 직접 디자인할 수 있는 유틸리티 우선 CSS 프레임워크.
  - [https://tailwindcss.com/docs](https://tailwindcss.com/docs)

---

## 2. ⚙️ 백엔드 (Backend) - 서버 및 비즈니스 로직

- **Node.js**: 배터리가 내장된 JavaScript 런타임.
  - [https://nodejs.org/en/docs/](https://nodejs.org/en/docs/)
- **Express.js**: Node.js를 위한 빠르고 유연한 웹 애플리케이션 프레임워크.
  - [https://expressjs.com/](https://expressjs.com/)
- **NestJS**: 효율적이고 확장 가능한 서버 사이드 Node.js 애플리케이션 구축 프레임워크.
  - [https://docs.nestjs.com/](https://docs.nestjs.com/)
- **Django**: 빠르고 깔끔한 웹 개발을 유도하는 고수준 Python 웹 프레임워크.
  - [https://docs.djangoproject.com/](https://docs.djangoproject.com/)
- **FastAPI**: Python 3.8+ 기반의 모던하고 매우 빠른 웹 API 프레임워크.
  - [https://fastapi.tiangolo.com/](https://fastapi.tiangolo.com/)
- **Spring Boot**: 독립 실행형이며 프로덕션 등급의 Spring 기반 애플리케이션 프레임워크 (Java/Kotlin).
  - [https://spring.io/projects/spring-boot](https://spring.io/projects/spring-boot)
- **Go (Golang)**: 신뢰할 수 있고 효율적인 소프트웨어 구축을 위한 오픈소스 프로그래밍 언어.
  - [https://go.dev/doc/](https://go.dev/doc/)
- **GraphQL**: API를 위한 쿼리 언어 및 런타임.
  - [https://graphql.org/learn/](https://graphql.org/learn/)
- **gRPC**: 구글에서 개발한 고성능 원격 프로시저 호출(RPC) 프레임워크.
  - [https://grpc.io/docs/](https://grpc.io/docs/)

---

## 3. 🗄️ 데이터베이스 및 캐싱 (Database & Caching)

- **PostgreSQL**: 강력하고 오픈소스인 객체-관계형 데이터베이스 시스템.
  - [https://www.postgresql.org/docs/](https://www.postgresql.org/docs/)
- **MySQL**: 세계에서 가장 인기 있는 오픈소스 RDBMS.
  - [https://dev.mysql.com/doc/](https://dev.mysql.com/doc/)
- **MongoDB**: 개발자와 애플리케이션을 위해 설계된 문서 지향 데이터베이스 (NoSQL).
  - [https://www.mongodb.com/docs/](https://www.mongodb.com/docs/)
- **Redis**: 대기 시간이 짧은 고성능 인메모리 데이터 스토어(캐시).
  - [https://redis.io/docs/](https://redis.io/docs/)

---

## 4. ☁️ 인프라, 클라우드 및 데브옵스 (Infra & DevOps)

- **AWS (Amazon Web Services)**: 포괄적이고 널리 채택된 클라우드 플랫폼.
  - [https://docs.aws.amazon.com/](https://docs.aws.amazon.com/)
- **GitHub Actions**: 개발 워크플로우를 자동화, 커스터마이즈, 실행하기 위한 CI/CD 도구.
  - [https://docs.github.com/en/actions](https://docs.github.com/en/actions)
- **Docker**: 컨테이너화된 애플리케이션을 빌드하고 실행하기 위한 플랫폼.
  - [https://docs.docker.com/](https://docs.docker.com/)
- **Kubernetes (K8s)**: 컨테이너화된 애플리케이션의 배포, 확장 및 관리를 자동화하는 오케스트레이션 시스템.
  - [https://kubernetes.io/docs/home/](https://kubernetes.io/docs/home/)
- **Nginx**: 고성능 HTTP 서버, 리버스 프록시, 로드 밸런서.
  - [https://nginx.org/en/docs/](https://nginx.org/en/docs/)

---

## 5. 🛠️ 보안 (Security) 및 인증

- **JWT (JSON Web Token)**: 정보(클레임)를 안전하게 전달하기 위한 기술의 개요.
  - [https://jwt.io/introduction](https://jwt.io/introduction)
- **OAuth 2.0**: 인증을 위한 산업 표준 프로토콜.
  - [https://oauth.net/2/](https://oauth.net/2/)
- **Web Security (MDN)**: 웹 보안의 기본 원칙과 가이드.
  - [https://developer.mozilla.org/en-US/docs/Web/Security](https://developer.mozilla.org/en-US/docs/Web/Security)
