Re:Boot 데이터 플로우 차트

AI 기반 커리어 빌드업 플랫폼의 전체 데이터 이동 흐름을 시각화합니다.

🏗️
1. 시스템 전체 아키텍처
Frontend - Backend - Database - External AI 데이터 흐름
Frontend → Backend: JWT Bearer Token 인증
Backend → DB: Django ORM CRUD
Backend ↔ AI: OpenAI API 양방향
강사 앱: 별도 포트 5174
🔐
2. 인증 및 사용자 데이터 플로우
회원가입 → 로그인 → JWT 토큰 → 인증 유지
🎙️
3. 실시간 STT 학습 데이터 파이프라인
음성 캡처 → Whisper STT → 저장 → AI 요약 → 퀴즈
📝
4. 퀴즈 채점 및 성취도 데이터 흐름
답안 제출 → 자동 채점 → 통계 → 대시보드
🏫
5. 수강 등록 및 클래스 관리
강사: 클래스 생성 + 코드 발급 / 학생: 코드 입력 + 등록
📊
6. 강사/매니저 대시보드 데이터 흐름
학생별 성취도 / 반 전체 성취율 관리
💼
7. 커리어 포트폴리오 및 AI 면접
학습 이력 → AI 포트폴리오 / 스킬블록 모의면접
🔄
8. 세션 복구 및 상태 유지
브라우저 새로고침 시 localStorage → Backend 동기화
🗂️
9. 데이터베이스 Entity 관계도
PostgreSQL: 사용자 → 세션 → STT → 요약 → 퀴즈 → 포트폴리오
⚡
10. 고급 기능 데이터 흐름
라이브 세션 / 수준 진단 / AI 챗봇 / 복습 루트 / 적응형 콘텐츠
라이브 세션: 실시간 강의 + 필기
수준 진단: 배치 테스트 → 갭 맵
AI 챗봇: RAG + LLM 응답
복습: 형성평가 → 간격 반복
적응형: AI 변형 → 교수자 승인
🔴 라이브 세션 — 실시간 수업 참여와 필기 공유

목적: 강사가 실시간 수업을 열면, 학생들이 즉시 참여하여 함께 학습하는 기능입니다.

데이터 흐름:

  • 강사가 POST /live/로 라이브 세션을 개설합니다.
  • 학생이 POST /live/join/으로 해당 세션에 입장합니다.
  • 수업 중 학생은 실시간으로 필기를 작성하고, 강사가 우수 필기를 승인(Approve)합니다.
  • 결석한 학생도 승인된 필기를 나중에 조회하여 수업 내용을 따라갈 수 있습니다.

핵심: 출석 학생의 필기 → 강사 검증 → 결석생 공유로, 모든 학생이 동일한 학습 기회를 가집니다.

🟠 수준 진단 — 배치 테스트와 갭 맵

목적: 학생의 현재 실력을 정확히 파악하고, 목표까지의 격차를 시각화하는 기능입니다.

데이터 흐름:

  • 학생이 배치 테스트(진단평가)를 응시합니다. → AI가 결과를 분석하여 현재 수준을 판별합니다.
  • 학생이 학습 목표(예: "React 중급")를 설정합니다.
  • 시스템이 갭 맵(Gap Map)을 생성합니다: "현재 수준"과 "목표 수준" 사이의 부족한 역량을 시각적으로 보여줍니다.

핵심: "내가 뭘 모르는지 모르는" 상태를 해소하고, 어디를 집중 학습해야 하는지 명확히 알려줍니다.

🔵 AI 튜터 챗봇 — RAG 기반 맞춤 답변

목적: 학생이 학습 중 궁금한 점을 AI에게 즉시 질문하고, 수업 내용에 기반한 정확한 답변을 받는 기능입니다.

데이터 흐름:

  • 학생이 질문을 입력합니다. → 시스템이 RAG(Retrieval-Augmented Generation)로 관련 학습 자료를 검색합니다.
  • 검색된 자료 + 질문을 LLM(GPT-4o)에 전달하여 맥락에 맞는 답변을 생성합니다.
  • 대화 내용은 DB에 저장되어, 나중에 다시 확인하거나 학습 분석에 활용됩니다.

핵심: 일반 ChatGPT와 달리, 실제 수업 자료(STT 로그, 요약본)를 참조하여 답변하므로 수업 맥락에 맞는 정확한 도움을 제공합니다.

🟣 복습 및 간격 반복 — 약점 기반 자동 복습

목적: 퀴즈에서 틀린 문제를 분석하여 약점을 파악하고, 과학적 간격 반복 학습으로 장기 기억을 돕는 기능입니다.

데이터 흐름:

  • 형성평가를 자동 생성하여 학생에게 제공합니다. → 학생이 제출하면 약점 분석(weak_zone_utils)이 실행됩니다.
  • 약점이 발견된 영역에 대해 복습 루트(Review Route)를 자동 생성합니다.
  • 에빙하우스 망각 곡선 기반의 간격 반복 스케줄에 따라 적절한 시점에 다시 문제를 출제합니다.

핵심: 학생이 직접 "뭘 복습할지" 고민할 필요 없이, 시스템이 자동으로 약한 부분을 반복 학습시켜 줍니다.

🟢 적응형 콘텐츠 — 수준별 맞춤 학습 자료

목적: 같은 수업을 듣는 학생이라도, 각자의 수준에 맞게 난이도가 다른 학습 자료를 제공하는 기능입니다.

데이터 흐름:

  • 교수자가 "이 자료를 수준별로 변형해달라"고 요청합니다. → GPT-4o가 초급/중급/고급 버전을 각각 생성합니다.
  • 교수자가 AI 생성 결과를 검토하고 승인(Approve)합니다. (AI 자동 배포가 아닌 교수자 검증 필수)
  • 승인된 콘텐츠가 학생의 진단 결과에 따라 해당 수준의 버전만 개별 배포됩니다.

핵심: 초급자에겐 쉬운 설명, 고급자에겐 심화 내용을 제공하여 "수업이 너무 어려워요/쉬워요" 문제를 해결합니다.
