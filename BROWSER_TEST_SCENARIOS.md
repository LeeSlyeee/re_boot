# 🧪 Re:Boot 브라우저 테스트 시나리오

> **목적**: 로컬 서버에서 실제 브라우저 조작을 통해 프론트-백엔드 연동을 검증
> **사전 준비**: 백엔드(8000) + 학생 프론트(5173) + 교수 프론트(5174) 서버 가동

---

## 📌 사전 준비

```bash
# 터미널 1: 백엔드
cd backend && source ../venv/bin/activate && python manage.py runserver 0.0.0.0:8000

# 터미널 2: 학생 프론트
cd frontend && npm run dev -- --port 5173

# 터미널 3: 교수 프론트
cd Professor_dashboard && npm run dev -- --port 5174
```

### 테스트 계정

| 역할 | 아이디       | 비밀번호      |
| :--- | :----------- | :------------ |
| 학생 | `testuser`   | `testpass123` |
| 교수 | `instructor` | `testpass123` |
