# Re:Boot Next Features Roadmap

현재 Re:Boot는 **학습 이탈 방지(Dynamic Re-routing)**와 **성취 증명(Skill Assetization)** 사이클을 완성하여, 개별 학습자의 성장에 최적화된 플랫폼이 되었습니다.

다음 단계로는 **사회적 상호작용(Social Interaction)**과 **실전 취업 역량(Practical Career Skill)**을 강화하는 기능을 제안합니다.

## 1. Gamification & Social Learning (동기 부여 강화)

### 1.1 Skill Leaderboard (스킬 랭킹)

- **개요**: "나만 뒤쳐지는 것 같다"는 불안감을 "나도 할 수 있다"는 경쟁심으로 전환.
- **기능**:
  - 주간/월간 스킬 블록 획득 랭킹.
  - "이번 주 우리 반 80%가 `Vue Router`를 획득했어요!" 알림.
  - 특정 스킬 조합(Set) 완성 시 **뱃지(Badge)** 부여 (예: `Front-End Master`).

### 1.2 Peer Review (동료 학습)

- **개요**: 혼자 공부하는 것이 아니라, 서로의 코드를 봐주며 성장.
- **기능**:
  - 내가 푼 퀴즈나 과제에 대해 다른 학생이 코멘트 달기.
  - "도움이 되었어요" 스티커 주고받기.

---

## 2. Advanced Career Services (취업 실전 대비)

### 2.1 AI Mock Interview (모의 면접)

- **개요**: 포트폴리오만으로는 부족한 '말하기' 능력 배양.
- **기능**:
  - 사용자의 **AI 포트폴리오**를 바탕으로 예상 면접 질문 생성.
  - **TTS(음성 합성)**로 질문하고, 학생의 답변을 **STT**로 분석하여 피드백.
  - "답변에서 `React`와의 차이점을 명확히 설명하지 못했습니다." 같은 구체적 조언.

### 2.2 Corporate Matching (기업 연계)

- **개요**: 기업이 원하는 인재를 '스킬 블록' 단위로 검색.
- **기능**:
  - 기업용 대시보드: "Vue.js + Python 스킬 보유자 검색" -> 해당 학생 익명 프로필 매칭.
  - **Reverse Recruiting**: 기업이 먼저 학생에게 면접 제안.

---

## 3. Mobile Experience (접근성 확대)

### 3.1 Push Notification (학습 알림)

- **개요**: PC 앞에 없어도 학습 흐름을 유지.
- **기능**:
  - "3일째 접속하지 않으셨네요. `Vue Component` 복습 한번 어때요?" 푸시 알림.
  - **Daily Quiz** 모바일 전용 UI 제공 (지하철 출퇴근길 학습).

### 3.2 Voice Coding Assistant (음성 코딩)

- **개요**: 키보드를 칠 수 없는 상황에서도 코딩 아이디어 정리.
- **기능**:
  - "파이썬으로 리스트 컴프리헨션 어떻게 쓰더라?" 음성 질문 -> 코드 스니펫 답변.
