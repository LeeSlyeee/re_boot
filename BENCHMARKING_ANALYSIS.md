## 1️⃣ 교육 기관용 LMS 분석

### 🟦 Canvas LMS (Instructure)

**핵심 강점:**

- **Intelligent Insights**: AI가 실시간으로 위험 학생을 식별하고 개입 시점을 제안
- **New Analytics**: 학생별 성적/출석/참여도를 시각적 대시보드로 제공
- **Video Analytics (Canvas Studio)**: 영상 시청 패턴 분석 (어디서 멈췄는지, 되감기 횟수)
- **IgniteAI Quiz Generation**: 강의 자료 기반 AI 퀴즈 자동 생성

**🎯 Re:Boot 적용 포인트:**

| 기능                     | 적용 방안                                                                                              | 기대 효과           |
| ------------------------ | ------------------------------------------------------------------------------------------------------ | ------------------- |
| **Intelligent Insights** | 교수자 대시보드에 "⚠️ 위험군 자동 감지" 알림 추가. 출석률 < 50%, 퀴즈 평균 < 40점인 학생을 자동 플래그 | 이탈 방지율 ↑       |
| **Video Analytics**      | 유튜브 학습 시 "되감기 구간" 분석 → 해당 부분 AI 보충 요약 자동 생성                                   | 취약 개념 자동 보강 |
| **Scheduled Feedback**   | 퀴즈 피드백을 즉시 공개하지 않고, 교수자가 "공개 시간"을 설정하는 옵션                                 | 커닝 방지           |
| **Portfolio Evidence**   | 포트폴리오에 "역량 증거 자료" 첨부 기능 (스킬블록 연동)                                                | 취업 연계 ↑         |

### 🟪 Blackboard Learn (Anthology)

**핵심 강점:**

- **Adaptive Release**: 특정 조건(성적, 날짜, 선수 과목 이수)을 충족해야 다음 콘텐츠에 접근 가능
- **Retention Center**: 학생의 위험도를 자동 계산하여 빨강/노랑/초록 신호등 표시
- **Mastery Learning**: 학습 목표별 숙달도 추적 및 최소 기준 미달 시 재수강 유도

**🎯 Re:Boot 적용 포인트:**

| 기능                               | 적용 방안                                                                           | 기대 효과        |
| ---------------------------------- | ----------------------------------------------------------------------------------- | ---------------- |
| **Adaptive Release (콘텐츠 잠금)** | 실라버스 주차별로 "이전 주차 퀴즈 60점 이상" 달성 시 다음 주차 자료 해제            | 학습 순서 보장   |
| **Retention Center**               | 교수자 대시보드에 학생별 **🔴🟡🟢 신호등 시스템** 도입. (출석 + 퀴즈 + 참여도 종합) | 직관적 위험 감지 |
| **Mastery Learning**               | 스킬블록 시스템에 "숙달도 70% 미만이면 재학습 추천" 로직 추가                       | 실력 보장        |

### 🟧 D2L Brightspace

**핵심 강점:**

- **Lumi AI Suite**: AI 튜터(Lumi Tutor), 학습 지원(Lumi Study Support), 성적 피드백(Lumi Grades)
- **LeaP (Adaptive Learning Engine)**: 학생별 "지식 맵"을 생성하고 개인화된 학습 경로 추천
- **Performance+ Dashboard**: 기관 전체의 학습 성과를 실시간 모니터링

**🎯 Re:Boot 적용 포인트:**

| 기능                         | 적용 방안                                                                                          | 기대 효과      |
| ---------------------------- | -------------------------------------------------------------------------------------------------- | -------------- |
| **Knowledge Map (지식 맵)**  | 현재 갭 맵을 "지식 맵"으로 확장: 스킬 간 의존 관계 시각화 (예: "클로저 이해" → "비동기 이해" 필요) | 학습 순서 안내 |
| **Lumi Study Support**       | 퀴즈 결과 기반으로 "보충 학습 자료"를 AI가 자동 추천 (현재 적응형 콘텐츠 시스템 확장)              | 개인화 ↑       |
| **Performance+ 전체 현황판** | 관리자(매니저) 전용 전체 클래스 비교 대시보드 추가                                                 | 기관 운영 효율 |

## 2️⃣ 온라인 코스 마켓플레이스 분석

### 🔵 Coursera

**핵심 강점:**

- **Professional Certificate**: 수료 시 산업 인정 자격증 발급 (Google, IBM 등)
- **🔥 스트릭(연속 학습)**: 매일 학습하면 스트릭 카운트가 올라가는 동기 부여 시스템
- **Peer Review (동료 평가)**: 학습자끼리 과제를 평가하는 상호 학습 모델

**🎯 Re:Boot 적용 포인트:**

| 기능               | 적용 방안                                                                                    | 기대 효과   |
| ------------------ | -------------------------------------------------------------------------------------------- | ----------- |
| **🔥 학습 스트릭** | 대시보드에 "연속 학습일" 카운터 추가 (출석/퀴즈/학습 기준). 3일, 7일, 30일 달성 시 특별 배지 | 습관 형성   |
| **🎓 수료증 발급** | 클래스 종강 + 전체 퀴즈 통과 시 "수료증 PDF" 자동 생성 (강사명, 학습 시간, 달성률 포함)      | 이력서 활용 |
| **📝 동료 평가**   | 프로젝트 과제 제출 후, 같은 클래스 학생 2~3명이 상호 리뷰                                    | 협업 능력 ↑ |
| **학점 연계**      | 포트폴리오에 "역량 인증서"를 PDF로 다운로드 가능하게 제공                                    | 공식 증명   |

## 3️⃣ 즉시 적용 가능한 Quick Win (난이도 🟢)

### 1. 🔥 학습 스트릭 (연속 출석 카운터)

```
현재: 출석률만 %로 표시
개선: "🔥 7일 연속 학습 중!" 카운터 + 스트릭 배지
구현: StudentChecklist 기반으로 연속 일수 계산
```

### 2. 🏆 배지/업적 시스템

```
현재: 스킬블록만 존재
개선: 행동 기반 배지 시스템
  - 🌱 첫 발걸음 (첫 수업 완료)
  - 🔥 불꽃 스트릭 (7일 연속)
  - 📚 다독왕 (학습 시간 50시간 돌파)
  - 💯 퀴즈 마스터 (퀴즈 만점 3회)
  - 🎓 코스 컴플리트 (클래스 수료)
```

### 3. 🎓 수료증 PDF 자동 생성

```
현재: 포트폴리오만 존재
개선: 클래스 종강 시 "수료증" PDF 자동 생성
  - 학생명, 클래스명, 강사명
  - 출석률, 퀴즈 평균, 학습 시간
  - 스킬블록 획득 목록
  - QR코드 (검증용)
```

### 4. 📢 클래스 공지사항 게시판

```
현재: 교수자 → 학생 메시지만 존재
개선: 클래스별 공지/질문 게시판
  - 교수자: 공지 작성
  - 학생: 질문/댓글 작성
  - 알림: 새 공지 시 대시보드 배지
```

## 4️⃣ 결론: Re:Boot의 차별화 포인트

Re:Boot이 위 플랫폼들과 **이미 차별화**되는 부분:

| Re:Boot 고유 기능           | 경쟁 플랫폼 대비                                  |
| --------------------------- | ------------------------------------------------- |
| **실시간 STT 기반 학습**    | Canvas/Blackboard에는 없음                        |
| **AI 자동 요약 + 퀴즈**     | D2L Lumi와 유사하나, 실시간 방식은 유일           |
| **스킬블록 + 갭 맵**        | LinkedIn Learning의 스킬 분석과 유사하나, 더 세밀 |
| **포트폴리오 자동 생성**    | Coursera 수료증보다 더 포괄적 (학습 데이터 기반)  |
| **적응형 콘텐츠 (Level별)** | D2L LeaP과 유사한 개인화                          |

**위 분석을 종합하면, Re:Boot이 가장 시급히 도입해야 할 기능은:**

> 1. **게이미피케이션** (스트릭 + 배지 + 리더보드)
> 2. **위험 학생 조기 경보 시스템** (신호등 + 자동 알림)
> 3. **수료증/역량 인증서 발급** (PDF + 공유)

이 세 가지는 **구현 난이도가 낮으면서도 사용자 경험에 큰 임팩트**를 줍니다.
